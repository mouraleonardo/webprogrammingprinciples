HTML5 Tags

Basic Tags

HTML5 introduces several new semantic elements that help structure web content. Here are some of the basic tags:

<!DOCTYPE>

The declaration defines the document type and version of HTML being used. For HTML5, it is written as:

<!DOCTYPE html>

<html>

The <html> tag is the root element of an HTML page. It wraps all the content on the page.

<html lang="en">
    <head>...</head>
    <body>...</body>
</html>

<head>

The <head> element contains meta-information about the document, such as its title, character set, and linked resources.

<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>

<body>

The <body> tag contains the content of the document, such as text, images, and other media.

<body>
    <h1>Hello, World!</h1>
    <p>Welcome to my webpage.</p>
</body>

Text Content Tags

HTML5 provides various tags for structuring text content:

<h1> to <h6>

Heading tags range from <h1> (most important) to <h6> (least important).

<h1>Main Heading</h1>
<h2>Sub Heading</h2>

<p>

The <p> tag defines a paragraph of text.

<p>This is a paragraph.</p>

<blockquote>

The <blockquote> tag is used for long quotations.

<blockquote>
    <p>This is a quote from someone important.</p>
</blockquote>

<strong> and <em>

The <strong> tag emphasizes text with strong importance, while <em> emphasizes text with normal importance.

<strong>Important text</strong>
<em>Emphasized text</em>

Structural Tags

HTML5 introduces new structural tags to enhance the semantic meaning of web pages:

<header>

The <header> tag represents introductory content, typically containing headings, logos, and navigation.

<header>
    <h1>Website Title</h1>
    <nav>...</nav>
</header>

<nav>

The <nav> tag defines a set of navigation links.

<nav>
    <a href="home.html">Home</a>
    <a href="about.html">About</a>
</nav>

<main>

The <main> tag specifies the main content of a document, excluding headers, footers, and sidebars.

<main>
    <article>...</article>
</main>

<footer>

The <footer> tag defines a footer for a document or section.

<footer>
    <p>Copyright Â© 2024</p>
</footer>

Media Tags

HTML5 enhances support for audio and video content:

<audio>

The <audio> tag is used to embed sound content in documents.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

<video>

The <video> tag is used to embed video content.

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Interactive Elements

HTML5 introduces new interactive elements:

<canvas>

The <canvas> tag is used to draw graphics on the fly via scripting (usually JavaScript).

<canvas id="myCanvas" width="200" height="100"></canvas>

<details> and <summary>

The <details> tag creates a disclosure widget from which the user can obtain additional information or controls. The <summary> tag provides a summary or title for the details.

<details>
    <summary>More information</summary>
    <p>Here is some more information.</p>
</details>

Form Elements

HTML5 includes several new input types for forms:

<input> Types

New input types include:

  • email: Validates the input to ensure it's an email address.
  • url: Validates the input to ensure it's a URL.
  • date: Provides a date picker.
  • range: Creates a slider control.

Semantic Tags

Semantic tags give meaning to the web content and improve accessibility:

<article>

The <article> tag specifies independent, self-contained content.

<article>
    <h2>Article Title</h2>
    <p>This is the content of the article.</p>
</article>

<section>

The <section> tag defines a section of content, typically with a heading.

<section>
    <h2>Section Title</h2>
    <p>This is a section of content.</p>
</section>

<aside>

The <aside> tag represents content related to the surrounding content, like a sidebar.

<aside>
    <p>This is an aside content.</p>
</aside>

Table Tags

HTML5 tables allow for structured data presentation:

<table>

The <table> tag is used to create a table.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

Conclusion

HTML5 brings a wealth of new tags and features that enhance the semantic meaning and functionality of web pages. Understanding these tags allows for better structuring of content and improved accessibility.
