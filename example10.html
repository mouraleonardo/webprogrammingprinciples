HTML Tables Tutorial

Basic HTML Table Structure

HTML tables are created using the following tags:

  • <table> - defines a table
  • <tr> - defines a table row
  • <td> - defines a table cell
  • <th> - defines a table header 
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>City</th>
    </tr>
    <tr>
        <td>John Doe</td>
        <td>25</td>
        <td>New York</td>
    </tr>
    <tr>
        <td>Jane Smith</td>
        <td>30</td>
        <td>London</td>
    </tr>
</table>

Table Borders

Borders can be added using CSS:

table {
    border-collapse: collapse;
    width: 100%;
}
table, th, td {
    border: 1px solid black;
    padding: 8px;
}
th {
    background-color: #f2f2f2;
}

Table Sizes

Tables can be sized using various CSS properties:

table {
    width: 75%;    /* Table width */
    margin: auto;  /* Center the table */
}
td {
    height: 50px;  /* Cell height */
}
.custom-width {
    width: 150px;  /* Custom column width */
}

Padding & Spacing

Control cell spacing using padding and border-spacing:

table {
    border-spacing: 10px;
    border-collapse: separate;
}
td {
    padding: 15px;
    background-color: #f8f9fa;
    border: 1px solid #dee2e6;

Colspan & Rowspan

Cells can span multiple columns or rows:

<table>
    <tr>
        <th colspan="2">Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>First</td>
        <td>Last</td>
        <td rowspan="2">25</td>
    </tr>
    <tr>
        <td colspan="2">Contact Info</td>
    </tr>
</table>
Name Age
First Last 25
Contact Info

Table Styling

Tables can be styled with various CSS properties:

table {
    border-collapse: collapse;
    width: 100%;
    box-shadow: 0 0 20px rgba(0, 0, 0, 0.1);
}
th {
    background-color: #009879;
    color: #ffffff;
    text-align: left;
}
th, td {
    padding: 12px 15px;
}
tbody tr {
    border-bottom: 1px solid #dddddd;
}
tbody tr:nth-of-type(even) {
    background-color: #f3f3f3;
}
tbody tr:last-of-type {
    border-bottom: 2px solid #009879;
}

Table Colgroup

The colgroup element is used to style specific columns. It allows you to apply styles to entire columns without having to specify styling for individual cells:

<table>
    <colgroup>
        <col style="background-color: #f0f0f0">
        <col style="background-color: #e0e0e0">
        <col style="background-color: #d0d0d0">
    </colgroup>
    <tr>
        <th>Product</th>
        <th>Price</th>
        <th>Stock</th>
    </tr>
    <tr>
        <td>Laptop</td>
        <td>$999</td>
        <td>25</td>
    </tr>
    <tr>
        <td>Phone</td>
        <td>$699</td>
        <td>50</td>
    </tr>
</table>

Additional Colgroup Features:

<table>
    <colgroup>
        <col span="2" style="background-color: #f0f0f0">  /* Spans two columns */
        <col style="background-color: #e0e0e0; width: 150px">  /* Sets width */
    </colgroup>
    /* Table content */
</table>

Common Colgroup Use Cases:

  • Styling alternate columns for better readability
  • Setting specific column widths
  • Hiding columns with visibility: collapse
  • Applying consistent styling to entire columns

Note: Some CSS properties may not work with colgroup in all browsers. The most widely supported properties are:

  • background-color
  • background
  • border
  • width
  • visibility

Additional Resources and Tips

  • Always include table headers for better accessibility
  • Use appropriate semantic markup (thead, tbody, tfoot)
  • Consider mobile responsiveness when designing tables
  • Use appropriate spacing and padding for better readability
  • Add hover effects to make interactive tables more engaging
