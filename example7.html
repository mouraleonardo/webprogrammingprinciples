HTML Fundamentals

This document covers the basics of HTML, including essential tags, their usage, and differences between various attributes.

HTML Basics

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses a series of elements to structure content on the web.

<!DOCTYPE html>
<html lang="en">
    <head></head>
    <body></body>
</html>

HTML Elements

An HTML element consists of a start tag, content, and an end tag.

<p>This is a paragraph.</p>

HTML Attributes

Attributes provide additional information about an HTML element. They are always specified in the start tag.

<a href="https://www.example.com">Link</a>

HTML Headings

HTML headings range from <h1> to <h6>. <h1> is the highest level, while <h6> is the lowest.

<h1>Main Heading</h1>
<h2>Subheading</h2>

HTML Paragraphs

Paragraphs are defined using the <p> tag.

<p>This is a paragraph.</p>

HTML Styles

CSS can be used to style HTML elements. Styles can be applied inline, internally, or externally.

<p style="color: blue;">This text is blue.</p>

HTML Formatting

Text can be formatted using tags such as <b> for bold, <i> for italics, and <u> for underline.

<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined Text</u>

HTML Quotations

Quotations can be added using <blockquote> for block quotations and <q> for inline quotations.

<blockquote>This is a block quote.</blockquote>
<q>This is an inline quote.</q>

HTML Comments

Comments can be added using the following syntax. Comments are not displayed in the browser.

<!-- This is a comment -->

HTML Colors

Colors can be defined using names, HEX, RGB, or HSL values.

<p style="color: red;">Red Text</p>
<p style="color: #00ff00;">Green Text</p>

HTML CSS

CSS (Cascading Style Sheets) is used to style HTML elements. It can be applied directly in HTML, in a <style> tag, or in an external stylesheet.

<style>
    p { color: blue; }
</style>

HTML Links

Links are created using the <a> tag. The href attribute specifies the URL of the page the link goes to.

<a href="https://www.example.com">Visit Example</a>

HTML Images

Images are embedded using the <img> tag, with the src attribute defining the image source.

<img src="image.jpg" alt="Description of Image">

HTML Favicon

A favicon is a small icon associated with a website. It can be added in the <head> section.

<link rel="icon" href="favicon.ico">

HTML Page Title

The page title is set using the <title> tag within the <head> section.

<title>My Web Page</title>

HTML Tables

Tables are created using the <table> tag, with <tr> for rows, <th> for header cells, and <td> for data cells.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

HTML Lists

Lists can be ordered (<ol>) or unordered (<ul>), with items defined using <li>.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

HTML Block & Inline

Block elements take up the full width available, while inline elements only take up as much width as necessary.

<div>This is a block element.</div>
<span>This is an inline element.</span>

HTML Div

The <div> tag is a block-level container for grouping elements and applying styles.

<div>Content goes here.</div>

HTML Classes

Classes are used to group elements for styling and can be applied to multiple elements.

<div class="my-class">Styled content</div>

HTML Id

The id attribute uniquely identifies an element on the page.

<div id="unique-element">Unique Content</div>

HTML Iframes

An <iframe> is used to embed another document within the current HTML document.

<iframe src="https://www.example.com"></iframe>

HTML JavaScript

JavaScript can be embedded using the <script> tag.

<script>
    alert("Hello, World!");
</script>

HTML File Paths

File paths can be absolute or relative. Absolute paths point to a specific location, while relative paths are based on the current document's location.

<a href="https://www.example.com">Absolute Link</a>
<a href="images/pic.jpg">Relative Link</a>

HTML Head

The <head> section contains meta-information about the document, such as title and linked resources.

<head>
    <title>Document Title</title>
    <link rel="stylesheet" href="styles.css">
</head>

HTML Layout

HTML provides the structure for web layouts, often using <div> and CSS for styling.

<div class="container">
    <header>Header</header>
    <nav>Navigation</nav>
    <main>Main Content</main>
    <footer>Footer</footer>
</div>

HTML Responsive

Responsive design ensures that web pages look good on all devices. Use CSS media queries and flexible layouts.

@media (max-width: 600px) {
    body { background-color: lightblue; }
}

HTML Computercode

Computer code can be displayed using the <pre> and <code> tags.

<pre>
function hello() {
    console.log("Hello, World!");
}
</pre>

HTML Semantics

Semantic HTML uses HTML markup to reinforce the meaning of the content. It improves accessibility and SEO.

<article>...</article>
<section>...</section>
<aside>...</aside>

HTML Style Guide

Consistent coding style improves readability. Use lowercase for tags and attributes, and properly indent your code.

<div class="container">
    <p>Paragraph text</p>
</div>

HTML Entities

HTML entities are used to display reserved characters. For example, use & for & and < for <.

<p>This <p> tag is a paragraph.</p>

HTML Symbols

Special symbols can be included using HTML entities, such as © for © and ® for ®.

<p>Copyright © 2024</p>

HTML Emojis

Emojis can be included in HTML using Unicode or directly via their character representation.

<p>I love programming! 😊</p>

HTML Charsets

Character encoding specifies how characters are represented. UTF-8 is the most common.

<meta charset="UTF-8">

HTML URL Encode

URL encoding replaces unsafe ASCII characters with a "%" followed by two hexadecimal digits.

https://www.example.com/search?q=hello%20world

HTML vs. XHTML

HTML is more forgiving with syntax, while XHTML is a stricter version of HTML, requiring all elements to be properly nested and closed.

<img src="image.jpg" alt="Image">  
<img src="image.jpg" alt="Image" />
