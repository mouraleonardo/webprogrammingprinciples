GitHub User Fetcher

How to Run This Code

  1. Download or clone the project files to your local machine.
  2. Ensure you have the following files in your project folder:
    • index.html
    • fetchUser.js
  3. To avoid CORS issues, use a local server to run the project. Here are a few options:
    • Using Live Server (VS Code Extension):
      • Install the Live Server extension from the Visual Studio Code marketplace.
      • Open your index.html file in VS Code.
      • Right-click on the HTML file and select Open with Live Server.
    • Using http-server (Node.js package):
      • Make sure you have Node.js installed on your computer.
      • Open your terminal/command prompt and navigate to your project folder.
      • Run npx http-server (you may need to install it with npm install -g http-server first).
      • Open your browser and go to http://localhost:8080 (or the specified port).
  4. Enter a GitHub username in the input field and click Fetch User Info to retrieve user information.
